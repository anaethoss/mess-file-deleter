### Delete specified file from folder

A simple node.js script to delete any specified (with file format) file from a folder.
This script can help to delete a specified files from a folder where has many files together.
It will delete all files that specified.

To specify files and folders, it need to change these two lines in the script:
```javascrpt
const folderPath = './'; // folder path
const fileFormat = 'vtt'; // file format
```
